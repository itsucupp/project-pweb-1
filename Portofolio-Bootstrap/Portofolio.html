Tentang Saya

Saya M. Yusup Supriadi, seorang mahasiswa yang antusias dalam dunia teknologi dan pemrograman. Berasal dari Majalengka, saya memiliki minat mendalam terhadap pengembangan perangkat lunak dan desain web. Saya memiliki keahlian di bidang pengembangan front-end menggunakan HTML, CSS, dan JavaScript. Selain itu, saya juga berpengalaman dalam menggunakan framework seperti Bootstrap untuk membangun website yang responsif dan menarik.

Saya telah mengerjakan beberapa proyek, termasuk membuat website portofolio pribadi dan aplikasi sederhana berbasis web. Tujuan saya adalah menjadi seorang pengembang perangkat lunak profesional yang dapat memberikan solusi teknologi untuk berbagai masalah. Saat ini, saya juga sedang mempelajari teknologi backend untuk memperluas keahlian saya.
